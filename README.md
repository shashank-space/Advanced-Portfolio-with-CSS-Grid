# Advanced Portfolio Website

A modern, responsive portfolio website built using **HTML5, CSS3, and JavaScript**. This project demonstrates advanced front-end development concepts including CSS Grid, Flexbox, CSS Variables, Animations, Responsive Design, Dark Mode, and BEM methodology.

---

## Features

### Layout & Design

* Responsive Design
* Mobile-Friendly Interface
* CSS Grid Layout System
* Flexbox Navigation and Layouts
* Modern UI Design

### CSS Features

* CSS Variables (Custom Properties)
* Dark & Light Theme Support
* CSS Animations
* CSS Transitions
* Hover Effects
* Focus Effects
* Advanced Selectors
* Pseudo-elements (`::before`, `::after`)
* BEM Naming Methodology

### JavaScript Features

* Theme Switcher
* Local Storage Theme Persistence
* Smooth Scrolling Navigation
* Scroll Reveal Animation
* Active Navigation Highlight
* Dynamic Greeting Message
* Interactive Project Cards

### Additional Sections

* About Section
* Skills Section
* Projects Section
* Gallery Section
* Contact Form
* Footer Section

## Concepts Demonstrated

### CSS Grid

Used for:

* Skills Layout
* Projects Layout
* Gallery Layout

### Flexbox

Used for:

* Navigation Bar
* Footer
* Contact Form Layout

### CSS Variables

Used for:

* Theme Management
* Colors
* Reusable Design Values

### CSS Animations

Used for:

* Fade-In Effects
* Scroll Reveal Effects
* Card Hover Effects
* Image Hover Effects

### BEM Methodology

Example:

```html
<div class="project-card project-card--featured">

    <h3 class="project-card__title">
        Portfolio Website
    </h3>

</div>
```

* Block → project-card
* Element → project-card__title
* Modifier → project-card--featured

## How to Run

1. Download or clone the project.
2. Open the project folder.
3. Open `index.html` in a browser.

OR

Use VS Code Live Server:

1. Install Live Server Extension.
2. Right Click `index.html`.
3. Click **Open with Live Server**.
